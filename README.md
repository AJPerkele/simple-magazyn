# System Magazynowy i Sprzedaży

Prosta aplikacja do zarządzania magazynem, zakupami i sprzedażą z interfejsem graficznym.

## Funkcje
- Zarządzanie produktami (dodawanie, usuwanie)
- Zakupy z alokacją FIFO
- Sprzedaż z automatycznym obliczaniem kosztów
- Inwentaryzacja magazynu
- Raporty w CSV i Excel
- Kursy EUR z NBP

## Instalacja

### Windows
